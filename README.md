## Installation

Oncelikle projeyi clonelayin. (Buraya sizin reponuzdan aldiginiz link gelecek)

```
git clone https://https://github.com/halibal/kodluyoruzilkrepo.git
```

## Usage

cd kodluyoruzilkrepo
